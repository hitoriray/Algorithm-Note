# 最长上升子序列模型（LIS）

原型：[最长上升子序列](../算法基础/18线性DP+区间DP.md)

[最长上升子序列原题链接](https://www.acwing.com/problem/content/897/)

- 代码

```c++
#include <iostream>
#include <algorithm>
using namespace std;
const int N = 1010;

int n;
int a[N], f[N];

int main()
{
    cin >> n;
    for (int i = 1; i <= n; i ++ ) cin >> a[i];
    int res = 0;
    for (int i = 1; i <= n; i ++ )
    {
        f[i] = 1;
        for (int j = 1; j < i; j ++ )
       		if (a[j] < a[i])
                f[i] = max(f[i], f[j] + 1);
    	res = max(res, f[i]);
    }
    cout << res << endl;
    
    return 0;
}
```
